An autonomous swarm of micro flying robots with range-based relative localization

This work proposes an autonomous 3D multi-robot system without external GPS or motion capture systems. The relative position and bearing between any two flying robots are estimated by onboard EKF, which takes as input sensory information (velocity, yaw rate, height, and ranging from Ultra wide-band sensor). With the estimated relative states, experiments are conducted on 5 Crazyflie, including formation flight, and leader-follower coordination while the leader has a monocular camera to detect and fly towards the target.

Real-world flights

  1. The first figure: a formation flight of 5 micro flying robots based on the relative localization.
  2. The second figure: a leader-follower flight through a window autonomously, where the follower coordinates based on the relative information.

The experiment code is based on Crazyflie autopilot software, and has following developed features.

  1. Two-way-ranging between any two flying robots.
  2. Relative localization using EKF.
  3. Formation control, and leader-follower control.

In the leader-follower test, the first Crazyflie is equipped with a monocular camera for autonomous flight.

Simulation

Simulation code is built with Python, in which any number of robots can be simulated to validate the proposed relative localization method.
