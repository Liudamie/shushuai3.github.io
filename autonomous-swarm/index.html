This work proposes an autonomous 3D multi-robot system without external GPS or motion capture systems. The relative position and bearing between any two flying robots are estimated by onboard EKF, which takes as input sensory information (velocity, yaw rate, height, and ranging from Ultra wide-band sensor). With the estimated relative states, experiments are conducted on 5 Crazyflie, including formation flight, and leader-follower coordination while the leader has a monocular camera to detect and fly towards the target.

Real-world flights

  1. The 1st figure: a formation flight of 5 micro flying robots.
  2. The 2nd figure: a leader-follower flight through a window autonomously.

Demo on commercial quadrotors off-the-shelf:

  1. Prepare multiple Crazyflies with loco deck and flow deck;
  2. Set the address of each crazyflie ending by E5, E6, E7, ...;
  3. Download source code and checkout the swarm branch;
  4. Set the NUM_CFs in lpsTwrTag.h to the actual number of crazyflies.
  5. In cfclient, connect to 'E5' crazyflie, and set keep_flying to 1.

In the leader-follower test, the leader Crazyflie is equipped with a monocular camera for autonomous flight.

Simulation

Simulation code is built with Python, in which any number of robots can be simulated to validate the proposed relative localization method.
